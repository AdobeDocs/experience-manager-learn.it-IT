# Snapshot Adobe {#snapshot}

Adobe Snapshot è uno strumento di dimostrazione visiva, progettato per illustrare la potenza di Dynamic Media per la distribuzione di immagini ottimizzate e dinamiche. Sperimenta immagini di test o URL di elementi multimediali dinamici per osservare visivamente l’output di vari modificatori di immagini Dynamic Media, ritagli avanzati e ottimizzazioni di Smart Imaging per le dimensioni del file (con distribuzione WebP e AVIF).
