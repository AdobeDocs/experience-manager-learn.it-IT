# Utilizzo di Dynamic Media con i componenti core di AEM Sites {#dynamic-media-sites-core-components}

Il componente Immagine, che fa parte dei componenti core Experience Manager, dispone del supporto integrato di Dynamic Media. Il componente Immagine consente all’autore di contenuto di utilizzare le funzioni di Dynamic Media come Predefiniti immagine, Ritaglio avanzato e Modificatori immagine su una pagina AEM Sites.
