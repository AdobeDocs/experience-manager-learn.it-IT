# Utilizzo  Adobe Cloud Manager - Ambienti

[Gli ](https://experienceleague.adobe.com/docs/experience-manager-cloud-manager/using/how-to-use/manage-your-environment.html) ambienti Cloud Manager sono composti dai servizi AEM Author, AEM Publish e Dispatcher. Diversi ambienti supportano i ruoli e possono essere coinvolti utilizzando diverse pipeline CI/CD. In genere, gli ambienti di Cloud Manager sono caratterizzati da un ambiente di produzione, un ambiente Stage e un ambiente di sviluppo.
